# Get Started with RunAgents

RunAgents is the platform for deploying AI agents that act securely. Whether you are building internal automation, customer-facing AI features, or multi-agent workflows -- we have got you covered.

Identity propagation, policy-driven access control, OAuth consent, just-in-time approvals, and a unified LLM gateway -- all managed for you.

---

## Start a Free Trial

Get full platform access with no upfront commitment.

Email **[try@runagents.io](mailto:try@runagents.io?subject=RunAgents Trial Request)** with:

- Your name
- Company
- Use case (brief description)
- Expected number of agents

We will set you up within 24 hours.

[Start Your Free Trial](mailto:try@runagents.io?subject=RunAgents Trial Request){.cta-button}

## Self-Hosted

Need to run RunAgents in your own infrastructure? We support deployment to any environment -- your cloud account, on-premise, or air-gapped networks. You retain full data sovereignty.

Email **[try@runagents.io](mailto:try@runagents.io?subject=Self-Hosted RunAgents)** with "Self-Hosted" in the subject line. We will provide a deployment guide, container image access, and onboarding support.

See the [Self-Hosted Deployment Guide](self-hosted/deployment.md) for an overview of what is included.
