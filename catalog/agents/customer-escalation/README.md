# Customer Escalation Agent

A customer operations starter that assembles a clear escalation brief for high-priority issues.

## What it includes
- Suggested model: `gpt-4.1`
- Required tools: `crm`, `ticketing`, `knowledge-base`, `chat`
- Default policies: `support-readonly`, `customer-data-guardrails`

## Deployment intent
Use this as a governed starting point for support escalations, account-risk handoffs, and cross-functional incident coordination.
