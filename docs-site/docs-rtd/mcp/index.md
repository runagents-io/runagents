# MCP Overview

The RunAgents MCP server exposes assistant-safe tools backed by the same Python client used by the SDK.

Install with:

```bash
pip install runagents[mcp]
```

Command:

```bash
runagents-mcp
```

The MCP server is intended for editors and coding assistants that can launch local MCP subprocesses and provide RunAgents workspace credentials through environment variables.
