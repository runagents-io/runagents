# SDK reference

The RunAgents Python SDK combines resource management, agent helpers, CLI tooling, MCP support, and runtime helpers in one package.

## Start with the SDK overview

- [Python SDK](../sdk/index.md)

## What the SDK covers

- `Client` for agents, tools, runs, approvals, and deploy flows
- `Agent` helpers for governed tool and model calls
- `@tool` decorators and runtime helpers
- shared configuration with the CLI
- MCP support for AI coding assistants

If you want a terminal-first path instead, use the [CLI reference](cli-reference.md).
