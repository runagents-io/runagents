# Executive Briefing Agent

A leadership productivity starter that assembles a concise daily briefing from meetings, project updates, and stakeholder signals.

## What it includes
- Suggested model: `gpt-4.1`
- Required tools: `calendar`, `project-tracker`, `knowledge-base`, `chat`
- Default policies: `leadership-readonly`, `internal-docs-readonly`

## Deployment intent
Use this as a governed starting point for executive morning briefs, staff prep, and leadership decision digests.
