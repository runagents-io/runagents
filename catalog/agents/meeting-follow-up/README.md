# Meeting Follow-Up Agent

A productivity starter that turns meeting notes into clear decisions, action items, and a ready-to-send recap.

## What it includes
- Suggested model: `gpt-4.1-mini`
- Required tools: `calendar`, `docs`, `project-tracker`, `chat`
- Default policies: `meeting-readonly`, `task-write-approval`

## Deployment intent
Use this as a governed starting point for turning meetings into execution-ready follow-up across teams.
